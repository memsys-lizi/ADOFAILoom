# 基础设计原则

## 1. 先写视觉意图，再写事件

每个段落先用一句话描述画面，而不是列事件。例如：

> 镜头缓慢推近，远景向外漂移，轨道保持稳定；两个强调点使用低透明度冷色前景闪，结尾黑场切到下一场景。

这句话应明确：

- 视觉主体是谁。
- 观看距离和方向如何变化。
- 哪一层在运动。
- 色彩如何变化。
- 哪些点是主强调，哪些只是过渡。

如果一句话中出现三个互相竞争的主动作，应先删减设计。

## 2. 可读性优先级

正常情况下使用以下优先级：

1. 当前与后续轨道。
2. 行星和判定位置。
3. 镜头运动方向。
4. 主题装饰。
5. 氛围装饰与后期。

前景装饰、Flash、Bloom、Blur、Glitch 和 ScreenTile 都可能破坏前两项。任何效果只要让玩家无法稳定定位轨道，就必须降低强度、缩短时间、移到背景层或删除。

## 3. 三层空间

一个稳定的视觉场景通常至少区分三层：

| 层 | 作用 | 常见实现 |
|---|---|---|
| 背景 | 色场、环境、远景运动 | 正 depth、低 parallax、背景 Flash、缓慢装饰移动 |
| 中景 | 主题形状、建筑、粒子、叙事对象 | 中等 parallax、与镜头方向相关的位移 |
| 前景 | 遮罩、边框、冲击形状、转场幕布 | 负 depth、Camera/CameraAspect、短时出现 |

`depth >= 0` 进入背景层，`depth < 0` 进入前景层。不要只靠更大的 scale 伪造前景；正确的 depth、parallax 和对比度更重要。

## 4. 视觉强度阶梯

为每段选择一个强度等级，并保留升级空间：

| 等级 | 用途 | 建议同时活跃的主通道 |
|---|---|---|
| 0 静息 | 开场、留白、恢复 | 稳定镜头 + 静态层次 |
| 1 流动 | 普通段落 | 轻微运镜或单层装饰运动 |
| 2 强调 | 小段变化 | 镜头变化 + 一种光效或轨道动作 |
| 3 冲击 | 明确重拍/转场 | 镜头、轨道、Flash 中选两到三项 |
| 4 高潮 | 少数核心时刻 | 多层协同，但仍保留主体和结束状态 |

如果整首谱一直处于 3 或 4，高潮就不再存在。强效果后安排至少一个明显的回落区。

## 5. 对比比绝对强度更重要

可用的对比轴包括：

- 远景与近景。
- 静止与运动。
- 大范围慢动与小范围快动。
- 高饱和与低饱和。
- 明亮与暗场。
- 平稳镜头与瞬时切镜。
- 清晰画面与短暂失真。

一次只强调一到两个对比轴。不要同时大幅推镜、旋转、改色、抖动、模糊并闪白，除非这是全谱极少数的最高强度节点。

## 6. 色彩规则

每个段落建立小型色板：

- 一个背景主色。
- 一个轨道或主体色。
- 一个强调色。
- 黑或白作为结构色。

Bloom、Flash、轨道色和主要装饰应从同一色板取色。滤镜用于改变材质感或暂时破坏色板，不应无理由引入新的主色。

转场有三种可靠方式：

- 保持色相，改变明度或饱和度。
- 保持一个共同色，把另外两色替换。
- 用黑/白遮挡完成硬切，再建立新色板。

## 7. 动画语法

每个可见对象都应有完整生命周期：

```text
声明初始状态 → 入场 → 保持/循环 → 强调变化 → 退场 → 隐藏或复位
```

常见运动角色：

- `Out*`：快速进入、逐渐稳定，适合入场与冲击后的回弹。
- `In*`：逐渐加速离开，适合退场和蓄力。
- `InOut*`：连续位移、色彩过渡和长镜头。
- `Back`：具有设计感的越界回弹，适合轨道和图形，不适合频繁使用。
- `Elastic/Bounce`：风格极强，只给少数对象和明确主题。
- `Linear`：持续漂移、循环、瞬时设置；不应作为所有自然运动的默认答案。

## 8. 状态与复位

每段结束前检查：

- 相机 relativeTo、position、rotation、zoom 是否处于下一段预期基线。
- 临时滤镜是否降到中性后关闭。
- Bloom 是否先淡到低强度再禁用。
- 前景 Flash 是否回到透明。
- Shake 是否结束。
- 装饰是否退场或隐藏。
- 轨道 position、rotation、scale、opacity 是否恢复或有意延续。
- ScreenTile 是否回到 `(1,1)`，ScreenScroll 是否回到 `(0,0)`。

不能依赖“下一段大概会覆盖它”。每个临时状态都要有明确所有者和结束点。

## 9. 可访问性与舒适度

- 高频闪光、纯白全屏、强烈色相反转和快速缩放必须稀少。
- 连续闪光应降低 opacity，并在频率、亮度、持续时间中至少限制两项。
- 强震动不应覆盖需要精确读轨的长段。
- 失真滤镜和屏幕平铺应短时使用。
- 关键路径不能只靠高视觉设置下才会执行的效果表达。
